# Earguard 앱 기획서 — 06. 챌린지

---

## 6-1. 챌린지 메인

> **참조 화면**: screen_23, screen_24

### 화면 목적
게임화(Gamification) 요소를 통해 사용자의 청각 건강 관리 습관 형성.
레벨, 미션, 업적, 보상 포인트 시스템으로 구성.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ 챌린지                                   │  ← 헤더
│ 하루에 도전하고 나만의 습관을 만들어요     │
├─────────────────────────────────────────┤
│  Level 5            N일 연속             │
│  [레벨 프로그레스 바  ====----]           │
│  현재 XP: XXX / 다음 레벨: XXX XP        │
├─────────────────────────────────────────┤
│  오늘의 미션                              │
│  ┌──────────────────────────────────┐   │
│  │ 조용한 공간 찾기          +80P    │   │
│  │ 70dB 이하 구간으로 이동하기       │   │
│  │ [지금 시작하기 >]                │   │
│  └──────────────────────────────────┘   │
│  ┌──────────────────────────────────┐   │
│  │ ✓ 이어폰 볼륨 60% 이하  완료!+80P │   │
│  └──────────────────────────────────┘   │
├─────────────────────────────────────────┤
│  업적                                    │
│  [아이콘] 일주일 연속 기록   +500P  획득  │
│  [🔒]    청력 마스터        +5000P 도전중 │
├─────────────────────────────────────────┤
│  보상 상점                               │
│  모은 포인트로 보상을 교환하세요           │
│  [상점 들어가기 >]                       │
└─────────────────────────────────────────┘
```

---

### UI 구성

#### 상단 헤더
| 요소 | 설명 |
|---|---|
| 타이틀 | "챌린지" |
| 서브 텍스트 | "하루에 도전하고 나만의 습관을 만들어요" |

#### 레벨 카드
| 요소 | 설명 |
|---|---|
| 레벨 | 현재 레벨 (예: Level 5) |
| 연속 일수 | "N일 연속" 달성 표시 |
| 경험치 (XP) | 현재 XP / 다음 레벨 필요 XP |
| 레벨 프로그레스 바 | 경험치 진행률 |
| 레벨업 달성 조건 | 레벨별 XP 기준 (어드민 설정) |

#### 오늘의 미션 섹션
| 요소 | 설명 |
|---|---|
| 섹션 타이틀 | "오늘의 미션" |
| 미션 카드 리스트 | 미션명 + 설명 + 진행률 + 포인트 + 상태 버튼 |
| 완료 미션 | 체크 아이콘 + "완료! +XXP 획득" 표시 |
| 진행 중 미션 | 진행 바 + "지금 시작하기 >" 버튼 |
| 미완료 미션 | 진행 바 + "지금 시작하기 >" 버튼 |

**미션 예시 (어드민에서 관리)**
| 미션명 | 설명 | 획득 포인트 |
|---|---|---|
| 조용한 공간 찾기 | 70dB 이하의 구간으로 이동하기 | +80P |
| 이어폰 볼륨 60% 이하 | 하루 종일 볼륨을 낮게 유지 | +80P |
| 소음 데이터 10회 기록 | 현재 소음을 10회 측정하기 | 0/10 진행 |

#### 업적 섹션
| 요소 | 설명 |
|---|---|
| 섹션 타이틀 | "업적" |
| 달성 업적 카드 | 아이콘 + 이름 + 포인트 + "획득 완료" 표시 |
| 미달성 업적 카드 | 자물쇠 아이콘 + 이름 + 포인트 + "도전 중" 표시 |

**업적 예시**
| 업적명 | 포인트 | 조건 |
|---|---|---|
| 일주일 연속 기록 | +500P | 7일 연속 소음 기록 |
| 청력 마스터 | +5000P | 청력 검사 완료 후 우수 결과 |
| 소음 마스터 | +500P | 소음 데이터 N회 기록 |
| 완벽한 한 달 | +500P | 한 달 연속 소음 할당량 유지 |

#### 보상 상점 배너
| 요소 | 설명 |
|---|---|
| 타이틀 | "보상 상점" |
| 서브 텍스트 | "모은 포인트로 보상을 교환하세요" |
| 버튼 | "상점 들어가기 >" |

> 보상 상점 상세 화면은 현재 디자인에 없음 — 별도 기획 필요. (포인트로 무엇을 교환할 수 있는지 기획 필요)
