# Earguard 앱 기획서 — 03. 홈

---

## 3-1. 홈 메인

> **참조 화면**: screen_12

### 화면 목적
앱의 메인 대시보드. 현재 소음 수치, 일일 소음 할당량, 소음 안전 지도, 청력 나이, 힐링 사운드 미니 플레이어를 한 화면에서 제공.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ [Earguard 로고]                    [🔔] │  ← 상단 헤더
│ 안녕하세요, [닉네임] 님                   │
│ [컨디션 메시지]                           │  ← 인사 메시지
│                                         │
│  [측정 중 / 측정 안함 토글]               │  ← 측정 상태 조작
│              XX.X dB                    │  ← 실시간 소음 수치
│           [ 소음 상태 배지 ]              │  ← 안전 / 주의 / 위험
│                                         │
├─────────────────────────────────────────┤
│  일일 소음 할당량 / Noise Budget          │
│  [프로그레스 바]                          │
│  오늘 남은 소음 여유: XX%                 │
│  [블록형 잔여량 시각화]                   │
├────────────────────┬────────────────────┤
│  소음 안전 지도     │  청력 나이          │
│  (커뮤니티 지도     │  XX 세             │
│   연동)            ├────────────────────┤
│                    │  챌린지 스트릭      │
│                    │  N일 연속           │
├─────────────────────────────────────────┤
│  🎵 [곡명] 재생 중              [⊞] [▶] │  ← 마음챙김에서 재생 시에만 노출
├─────────────────────────────────────────┤
│  [홈] [기록] [마음챙김] [챌린지] [커뮤] [설정] │
└─────────────────────────────────────────┘
```

---

### UI 구성

#### 상단 헤더
| 요소 | 설명 |
|---|---|
| Earguard 로고 | 좌측 상단 |
| 알림 아이콘 | 우측 상단, 탭 시 알림 목록 이동 |

#### 인사 메시지
| 요소 | 설명 |
|---|---|
| 사용자 이름 인사 | "안녕하세요, [닉네임] 님" |
| 컨디션 메시지 | 소음 상태에 따른 동적 메시지 |

| 상태 | 메시지 |
|---|---|
| 안전 | "귀 건강을 위한 완벽한 하루에요" |
| 주의 | "소음이 조금 높아요. 잠깐 조용한 곳으로 이동해보세요" |
| 위험 | "지금 소음이 위험 수준이에요. 청각 보호가 필요합니다" |
| 미측정 | "오늘 하루 귀 건강을 챙겨보세요" |

#### 실시간 소음 측정 카드
| 요소 | 설명 |
|---|---|
| 측정 상태 토글 | "측정 중" / "측정 안함" 전환 버튼 — 이 카드 안에서 조작 |
| 소음 수치 | 측정 중: 현재 dB 실시간 표시 / 미측정: 마지막 측정값 + "마지막 측정" 표시 |
| 소음 상태 배지 | 측정 중일 때만 표시 — 안전 / 주의 / 위험 |

#### 일일 소음 할당량 카드 (Noise Budget)
> ⚠️ 안전 범위 100 기준 누적 계산식 ODSO 전달 예정 — 계산식 수신 후 수치 로직 및 잔여량/사용량 표현 방식 확정 (기록 > 소음 할당량 탭 동일)

| 요소 | 설명 |
|---|---|
| 카드 타이틀 | "일일 소음 할당량 / Noise Budget" |
| 프로그레스 바 | 오늘 소음 할당량 사용률 |
| 잔여량 텍스트 | "오늘 남은 소음 여유: XX%" |
| 블록형 잔여량 시각화 | 남은 여유분 블록으로 표시 |
| 탭 시 동작 | 기록 > 소음 할당량 탭으로 이동 |


#### 소음 안전 지도 카드
| 요소 | 설명 |
|---|---|
| 지도 미리보기 | 커뮤니티 탭 지도와 동일 데이터 — 거주지 주변 소음 요약 표시 |
| 탭 시 동작 | 상세보기 팝업 표시 |

**상세보기 팝업**
| 요소 | 설명 |
|---|---|
| 지도 확장 표시 | 거주지 주변 소음 핀 지도 |
| 소음 상승 위치 목록 | 최근 소음이 많이 오른 위치 순위 표시 |
| 닫기 버튼 | 팝업 닫기 |
| 커뮤니티 바로가기 | 커뮤니티 > 지도 탭으로 이동 |

#### 청력 나이 카드
| 요소 | 설명 |
|---|---|
| 수치 | 청력 나이 (예: 28세) |
| 연동 | 온보딩 청력 검사 결과 + 기록 > 정밀 자기검사 결과 기반으로 업데이트 |
| 탭 시 동작 | 기록 > 정밀 자기검사 탭으로 이동 |

#### 챌린지 스트릭 카드
> 💡 추가 기획 여지 있음 — 스트릭 표현 방식, 홈에서의 미션 인터랙션 등 확장 검토 필요

| 요소 | 설명 |
|---|---|
| 연속 일수 | N일 연속 달성 표시 |
| 진행 중 미션 요약 | 오늘 미션 간략 표시 |
| 탭 시 동작 | 챌린지 탭으로 이동 |

#### 힐링 사운드 미니 플레이어
| 요소 | 설명 |
|---|---|
| 노출 조건 | 마음챙김 탭에서 사운드 재생 중일 때만 표시 |
| 곡명 | 현재 재생 중인 곡명 |
| 재생/정지 버튼 | |
| 탭 시 동작 | 마음챙김 탭으로 이동 |

### 데이터 / 연동
| 데이터 | 연동 |
|---|---|

| 일일 소음 할당량 계산식 | ⏳ ODSO 제공 예정 |
| 소음 안전 지도 | 커뮤니티 탭 지도와 동일 데이터 |
| 청력 나이 | 온보딩 청력 검사 + 기록 > 정밀 자기검사 결과 |
| 힐링 사운드 | 마음챙김 탭 재생 상태 연동 |
