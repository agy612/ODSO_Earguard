## 5-1. 마음챙김 메인

> **참조 화면**: screen_19 (기분 미기록 상태), screen_20 (기분 기록 후)

> 마음챙김 탭은 2개 서브탭으로 구성: **[힐링]** | **[기분 기록]**

### 화면 목적
소음으로 인한 스트레스 관리 및 마음 건강 케어 허브 화면.
기분 기록 → AI 스트레스 분석 → AI 힐링 가이드 → 힐링 사운드 재생으로 이어지는 플로우.

---

### 화면 레이아웃

**기분 미기록 상태 (screen_19)**
```
┌─────────────────────────────────────────┐
│ 마음 챙김                                │  ← 헤더
│ [힐링] [기분 기록]                       │  ← 서브탭
│ 오늘 하루도 마음이 지쳤나요?               │

│  오늘 기분이 어때요?                      │

├─────────────────────────────────────────┤
│  힐링 사운드 라이브러리                   │
│  ┌─────────┐  ┌─────────┐              │
│  │썸네일   │  │썸네일   │              │

```


### UI 구성

#### 상단 헤더
| 요소 | 설명 |
|---|---|
| 타이틀 | "마음 챙김" |
| 서브 텍스트 | "오늘 하루도 마음이 지쳤나요? 소음으로 지친 마음을 풀어보세요" |

#### 오늘 기분 기록 카드
| 상태 | UI |
|---|---|
| 미기록 상태 (screen_19) | "오늘 기분이 어때요?" + 이모지 3종 선택 버튼 |
| 기록 완료 상태 (screen_20) | 선택된 기분 이모지 표시 + AI 분석 카드 노출 |

**기분 선택 항목**
| 이모지 | 레이블 |
|---|---|
| 😊 | 기분 좋아 |
| 😐 | 그냥 |
| 😠 | 별로야 |

> 기획 제안: 3단계보다 5단계 이모지(매우 좋음 / 좋음 / 보통 / 나쁨 / 매우 나쁨)로 확대하면 AI 분석 정확도 향상 가능. ODSO와 협의 필요.

#### AI 스트레스 분석 카드 (기분 기록 후 노출 — screen_20)
| 요소 | 설명 |
|---|---|
| 카드 타이틀 | "AI 스트레스 분석" |
| 서브 텍스트 | "소음 데이터 기반 AI 분석" |
| 스트레스 지수 | 수치 (예: 5) + 진행 바 |
| 상태 메시지 | 분석 결과 메시지 (예: "스트레스 수준이 낮습니다. 잘 관리하고 계세요!") |
| 자세히 보기 버튼 | AI 스트레스 분석 상세 팝업 오픈 |

#### AI 힐링 가이드 배너 (기분 기록 후 노출 — screen_20)
| 요소 | 설명 |
|---|---|
| 타이틀 | "AI 힐링 가이드" |
| 서브 텍스트 | "스트레스를 줄이는 맞춤 행동 가이드를 확인하세요" |
| 버튼 | "지금 시작하기 >" → AI 힐링 가이드 팝업 오픈 |

#### 힐링 사운드 라이브러리
| 요소 | 설명 |
|---|---|
| 섹션 타이틀 | "힐링 사운드 라이브러리" |
| 콘텐츠 카드 리스트 | 썸네일 + 제목 + 설명 + 재생 시간 + 재생 버튼 |
| 카드 탭 시 | 오디오 플레이어 화면으로 이동 |

**힐링 사운드 예시 항목 (마인드톤 콘텐츠 기반)**
| 제목 | 설명 | 시간 |
|---|---|---|
| 비 오는 날 | 자연을 통해서 치유하는 빗소리 | 3분 |
| 깊은 숲속 | 바람에 흔들리는 나뭇잎 소리 | 5분 |
| 별빛 고요 | 고요 속의 자연 시간을 담은 소리 | 3분 |
| 잔잔한 피아노 | 마음을 편안하게 하는 선율 | 3분 |

#### 하단 토스트 메시지
- 기분 기록 완료 시: "오늘의 기분이 기록되었습니다" 토스트 표시

## 5-2. AI 스트레스 분석 상세 팝업

> **참조 화면**: screen_21

### 화면 목적
AI 스트레스 지수의 세부 분석 요인 확인.

---

### 화면 레이아웃

```
┌──────────────────────────────────────┐
│  스트레스 분석 상세                    │
│  현재: 5       오늘 최고: 15           │
│  [=======                          ] │
│                                      │
│  소음 노출       +0                   │
│  감정 기록       +15                  │
│  날씨            +0                   │
│  활동량          +0                   │
│                                      │
│  [힐링 받으러 가기]                   │
└──────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 팝업 타이틀 | "스트레스 분석 상세" |
| 현재 스트레스 지수 | 좌측: 현재 값 / 우측: 오늘 최고값 |
| 측정 진행 바 | 전체 스케일 대비 현재 위치 |
| 분석 요인 리스트 | 각 요인별 기여 수치 표시 |
| 하단 버튼 | "힐링 받으러 가기" → AI 힐링 가이드 팝업 오픈 |

**스트레스 분석 요인 항목**
| 요인 | 예시 수치 |
|---|---|
| 소음 노출 | +0 |
| 감정 기록 | +15 |
| 날씨 | +0 |
| 활동량 | +0 |

> 기획 제안: 요인별 설명 툴팁 추가 (사용자가 왜 이 수치인지 이해할 수 있도록)

## 5-3. AI 힐링 가이드 팝업

> **참조 화면**: screen_22

> 💡 추가 기획 필요 — 기분 기록 데이터, 소음 노출 이력, 스트레스 분석 요인 등을 복합적으로 활용한 AI 개인화 가이드 로직 구체화 필요. 어떤 데이터를 어떤 방식으로 조합해 가이드를 생성할지 ODSO와 협의.

### 화면 목적
사용자의 스트레스 분석 결과 및 기분 기록 데이터를 바탕으로 맞춤 행동 가이드 제공.

---

### 화면 레이아웃

```
┌──────────────────────────────────────┐
│  AI 힐링 가이드                        │
│  맞춤 행동 가이드                      │
│                                      │
│  [아이콘] 조용한 공간 찾기              │
│           +38% 스트레스 감소           │
│  [아이콘] 자연 소리 듣기               │
│           +70% 이완                   │
│  [아이콘] 규칙적인 식사하기             │
│           +50% 에너지                 │
│  [아이콘] 충분한 수면으로 청력 회복     │
│  [아이콘] 천천히 호흡하기              │
└──────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 팝업 타이틀 | "AI 힐링 가이드" |
| 서브 텍스트 | "맞춤 행동 가이드" |
| 가이드 항목 리스트 | 아이콘 + 행동 제목 + 효과 설명 + 효과 수치 배지 |

**힐링 가이드 항목 예시**
| 행동 | 효과 | 배지 |
|---|---|---|
| 조용한 공간 찾기 | 가장 즉각적인 효과 | +38% 스트레스 감소 |
| 자연 소리 듣기 | 이완 효과 | +70% 이완 |
| 규칙적인 식사하기 | 에너지 회복 | +50% 에너지 |
| 충분한 수면으로 청력 회복 | 청력 민감도 회복 | |
| 천천히 호흡하기 | 즉각적 이완 | |

## 5-4. 기분 기록 탭

### 화면 목적
오늘의 기분을 체크하고, 과거 기분 기록을 캘린더 형식으로 조회.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ 마음 챙김                                │
│ [힐링] [기분 기록]                       │
├─────────────────────────────────────────┤
│  오늘 기분이 어때요?                      │  ← 오늘 미기록 시 표시
│  [😊 기분 좋아]  [😐 그냥]  [😠 별로야]  │
│  (기록 완료 시: 선택된 이모지 + 완료 표시) │
├─────────────────────────────────────────┤
│  2025년 4월                  [< >]      │  ← 월 전환
│  월  화  수  목  금  토  일              │
│   -   -   😊  😐  😠  😊   -           │  ← 날짜별 이모지 표시
│   😊  😐  😊  -   -   -   😠           │
│  ...                                    │
├─────────────────────────────────────────┤
│  이번 달 기록: N일                        │
│  가장 많은 기분: [이모지] XX회            │
└─────────────────────────────────────────┘
```

---

### UI 구성

#### 오늘 기분 체크
| 요소 | 설명 |
|---|---|
| 기분 선택 버튼 | 이모지 선택 (힐링 탭과 동일) |
| 기록 완료 상태 | 선택된 이모지 + 완료 표시 (재선택 가능 여부 ODSO 협의) |

#### 기분 캘린더
| 요소 | 설명 |
|---|---|
| 월 표시 + 전환 버튼 | 이전/다음 월 조회 |
| 날짜별 기분 이모지 | 기록된 날짜에 해당 이모지 표시, 미기록일은 빈칸 |
| 날짜 탭 시 | 해당 날짜 기분 상세 (이모지 + 당일 소음 데이터 요약) |

#### 월간 요약
| 요소 | 설명 |
|---|---|
| 이번 달 기록 일수 | 기분을 기록한 날짜 수 |
| 가장 많은 기분 | 해당 월 최다 선택 이모지 + 횟수 |
