# Earguard 앱 기획서 — 01. 로그인 / 회원가입

---

## 1-1. 스플래시 / 소셜 로그인 진입 화면

> **참조 화면**: screen_01

### 화면 목적
앱 최초 진입 시 표시되는 화면. 소셜 로그인 또는 이메일 로그인/회원가입으로 분기.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│                                         │
│         [앱 아이콘 + Earguard 로고]      │
│                                         │
│     당신의 동네는 얼마나 조용한가요       │
│  전국 소음건강지도와 함께 건강한          │
│  청각습관을 만들어보세요                  │
│                                         │
│  [카카오로 시작하기            ]         │
│  [Google 계정으로 시작하기     ]         │
│  [Apple 로그인하기             ]         │
│                                         │
│         로그인 / 회원가입                │
└─────────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 앱 아이콘 + 로고 | 중앙 상단, "Earguard" 텍스트 |
| 메인 문구 | "당신의 동네는 얼마나 조용한가요" |
| 서브 문구 | "전국 소음건강지도와 함께 건강한 청각습관을 만들어보세요" |
| 카카오로 시작하기 | 노란색 버튼, 카카오 로고 |
| Google 계정으로 시작하기 | 흰색 버튼, 구글 로고 |
| Apple 로그인하기 | 흰색 버튼, 애플 로고 |
| 로그인 / 회원가입 텍스트 링크 | 하단, 이메일 방식 화면으로 이동 |

---

## 1-2. 로그인 화면 (이메일)

> **참조 화면**: screen_02

### 화면 목적
이메일/비밀번호를 통한 자체 로그인.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│         [앱 아이콘 + 로고]               │
│  [로그인] [회원가입]                     │  ← 탭 전환
├─────────────────────────────────────────┤
│  이메일  [user@email.com              ] │
│  비밀번호 [••••••••              👁  ] │
│                                         │
│  [로그인]                               │
│                                         │
│           비밀번호 찾기                  │
└─────────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 앱 아이콘 + 로고 | 상단 |
| 탭: 로그인 / 회원가입 | 탭 전환 UI |
| 이메일 입력 필드 | placeholder: user@email.com |
| 비밀번호 입력 필드 | 마스킹 처리, 표시/숨김 토글 |
| 로그인 버튼 | |
| 비밀번호 찾기 링크 | 이메일로 재설정 링크 발송 |

---

## 1-3. 회원가입 화면 (이메일)

> **참조 화면**: screen_03

### 화면 목적
이메일 기반 신규 계정 생성.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│  [로그인] [회원가입]                     │
├─────────────────────────────────────────┤
│  이름     [앱 내 표시 이름             ] │
│  이메일   [                           ] │
│  비밀번호 [8자 이상              👁   ] │
│  비밀번호 확인  [                  👁  ] │
│                                         │
│  □ 이용약관 동의 (필수)                 │
│  □ 개인정보 처리방침 동의 (필수)         │
│  □ 위치 정보 이용 동의 (필수)           │
│                                         │
│  [회원가입하기]                          │
└─────────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 탭: 로그인 / 회원가입 | 회원가입 탭 선택 상태 |
| 이름 입력 필드 | 앱 내 표시 이름 |
| 이메일 입력 필드 | |
| 비밀번호 입력 필드 | 8자 이상, 영문·숫자·특수문자 포함 권장 |
| 비밀번호 확인 입력 필드 | |
| 이용약관 동의 체크박스 | 필수 동의 항목 (약관 내용 링크 포함) |
| 회원가입하기 버튼 | |

### 필요 약관
- 이용약관
- 개인정보 처리방침
- 위치 정보 이용 동의

> ⚠️ 이용약관 및 개인정보처리방침 내용 ODSO 제공 필요
