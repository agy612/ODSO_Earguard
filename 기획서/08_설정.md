# Earguard 앱 기획서 — 08. 설정

## 8-1. 설정 메인

> **참조 화면**: screen_31

### 화면 목적
계정 관리, 기기 연결, 앱 설정, 데이터·개인정보 설정의 진입점 화면.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ 설정                                     │
├─────────────────────────────────────────┤
│  [프로필 이미지]  Kakao 사용자           │
│                   user@email.com        │  ← 탭 시 프로필 수정
├─────────────────────────────────────────┤
│  계정 관리                               │
│  시리얼 번호 변경                    >   │
│  건강 설정                           >   │
├─────────────────────────────────────────┤
│  기기 연결                               │
│  Earguard 기기                       >   │
├─────────────────────────────────────────┤
│  앱 설정                                 │
│  알림                        [토글]     │
│  소음 공유                   [토글]     │
├─────────────────────────────────────────┤
│  데이터 및 개인정보                      │
│  데이터 서비스 (청각베이스라인)       >  │
│  데이터 서비스 관리                  >   │
│  개인정보 처리방침                   >   │
│  이용약관                            >   │
│  로그아웃                                │
│  회원 탈퇴                               │
└─────────────────────────────────────────┘
```

---

### UI 구성

#### 사용자 프로필 카드
| 요소 | 설명 |
|---|---|
| 프로필 이미지 | 소셜 로그인 프로필 사진 또는 기본 이미지 |
| 사용자 이름 | 예: "Kakao 사용자" |
| 이메일 | |
| 탭 시 | 프로필 수정 화면으로 이동 |

#### 계정 관리 섹션
| 항목 | 이동 |
|---|---|
| 시리얼 번호 변경 | 비밀번호 변경 화면 |
| 건강 설정 | 건강 정보 수정 화면 |

#### 기기 연결 섹션
| 항목 | 이동 |
|---|---|
| Earguard 기기 | 기기 설정 화면 |

#### 앱 설정 섹션
| 항목 | 타입 | 설명 |
|---|---|---|
| 알림 | 토글 | 푸시 알림 ON/OFF |
| 소음 공유 | 토글 | 측정 데이터 커뮤니티 자동 공유 ON/OFF |
| 기타 항목 | 토글/링크 | TBD |

#### 데이터 및 개인정보 섹션
| 항목 | 이동 |
|---|---|
| 데이터 서비스 (청각베이스라인) | 베이스라인 데이터 재설정 화면 |
| 데이터 서비스 관리 | 개인 데이터 관리 (다운로드, 삭제 등) |
| 개인정보 처리방침 | 웹뷰 또는 외부 링크 |
| 이용약관 | 웹뷰 또는 외부 링크 |
| 로그아웃 | 로그아웃 확인 팝업 → 스플래시 화면으로 이동 |
| 회원 탈퇴 | 탈퇴 확인 팝업 → 데이터 삭제 처리 |

## 8-2. 프로필 수정

> **참조 화면**: screen_32

### 화면 목적
사용자 닉네임, 나이, 프로필 이미지 수정.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ [←] 프로필 수정                          │
├─────────────────────────────────────────┤
│         [프로필 이미지 + 카메라 아이콘]   │
│         프로필 사진은 나만 볼 수 있어요   │
│                                         │
│  이메일  user@email.com  (수정 불가)     │
│  이름    [닉네임 입력               ]   │
│  나이    [숫자 입력                 ]   │
│                                         │
│  [저장하기]                              │
└─────────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 뒤로가기 버튼 | |
| 타이틀 | "프로필 수정" |
| 프로필 이미지 | 원형 이미지 + 카메라 아이콘 (탭 시 갤러리/카메라 선택) |
| 이메일 표시 | 수정 불가 (읽기 전용) |
| 프로필 정보 안내 | "프로필 사진은 나만 볼 수 있어요" |
| 이름 입력 필드 | 닉네임 변경 |
| 나이 입력 필드 | 숫자 입력 |
| 저장하기 버튼 | 변경 사항 저장 |

## 8-3. 비밀번호 변경

> **참조 화면**: screen_33

### 화면 목적
이메일 가입 사용자의 비밀번호 변경.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ [←] 비밀번호 변경                        │
├─────────────────────────────────────────┤
│  현재 비밀번호  [••••••••          👁]   │
│  새 비밀번호   [••••••••          👁]   │
│  새 비밀번호 확인  [••••••••       👁]   │
│                                         │
│  비밀번호는 영문 대/소문자, 숫자,         │
│  특수문자를 포함하는 것을 권장합니다      │
│                                         │
│  [비밀번호 변경하기]                     │
└─────────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 뒤로가기 버튼 | |
| 타이틀 | "비밀번호 변경" |
| 현재 비밀번호 입력 | 마스킹 처리 |
| 새 비밀번호 입력 | 8자 이상 |
| 새 비밀번호 확인 입력 | 일치 여부 실시간 표시 |
| 안내 문구 | "비밀번호는 영문 대/소문자, 숫자, 특수문자를 포함하는 것을 권장합니다" |
| 비밀번호 변경하기 버튼 | 모든 조건 충족 시 활성화 |

## 8-4. 건강 정보 수정

> **참조 화면**: screen_34

### 화면 목적
신체 정보 입력 및 스마트폰 건강 앱 연동. 청력 나이 계산, 소음 할당량 개인화에 활용.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ [←] 건강 설정                            │
├─────────────────────────────────────────┤
│  아이폰 건강 앱 또는 수동 요소를 설정해  │
│  건강 데이터를 불러올 수 있어요           │
│  [건강 데이터 가져오기]                  │
├─────────────────────────────────────────┤
│  키        [xxx cm  ]                   │
│  몸무게    [xxx kg  ]                   │
│  활동량    [낮음] [보통] [높음]          │
│                                         │
│  [저장하기]                              │
└─────────────────────────────────────────┘
```

---

### UI 구성

#### 스마트폰 건강 앱 연동 배너
| 요소 | 설명 |
|---|---|
| 설명 | "아이폰 건강 앱 또는 수동 요소를 설정해 건강 데이터를 불러올 수 있어요" |
| 연동 버튼 | "건강 데이터 가져오기" |

**iOS**: Apple Health (HealthKit) 연동
- 연동 시 키, 몸무게, 소음 노출 데이터 자동 동기화
- ⚠️ Apple Health의 환경 소음 항목 연동 가능 여부 사전 확인 필요

**Android**: Google Health Connect 연동 검토 필요

#### 직접 입력 섹션
| 요소 | 설명 |
|---|---|
| 키 입력 (cm) | 숫자 입력 |
| 몸무게 입력 (kg) | 숫자 입력 |
| 활동량 선택 | 낮음 / 보통 / 높음 선택 |
| 저장하기 버튼 | |

## 8-5. 기기 설정

> **참조 화면**: screen_35

### 화면 목적
연결된 Earguard 기기 관리, 소음 측정 주기 설정, 캘리브레이션.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ [←] 기기 설정                            │
├─────────────────────────────────────────┤
│  내 기기 찾기                             │
│  [디바이스 이름으로 검색            ]    │
│  [디바이스 탐색]                         │
├─────────────────────────────────────────┤
│  소리 저장 주기                           │
│  데이터가 기기에 저장되는 간격 설정        │
│  [10초] [30초] [1분(기본)] [5분]         │
├─────────────────────────────────────────┤
│  자동 기록 시작                           │
│  앱 실행 시 자동으로 데이터 저장 시작     │
│                              [토글]     │
├─────────────────────────────────────────┤
│  캘리브레이션                             │
│  정확한 소음 측정을 위한 보정 설정        │
│  [캘리브레이션 시작]  ← 기기 연결 시 활성 │
└─────────────────────────────────────────┘
```

---

### UI 구성

#### 내 기기 찾기
| 요소 | 설명 |
|---|---|
| 검색 입력 | "디바이스 이름으로 검색" |
| 디바이스 탐색 버튼 | BLE 스캔 시작 → 주변 기기 목록 표시 |

#### 소리 저장 주기
| 요소 | 설명 |
|---|---|
| 섹션 타이틀 | "소리 저장 주기" |
| 설명 | "데이터가 기기에 저장되는 간격 설정" |
| 선택지 | 10초마다 / 30초마다 / 1분마다(기본값) / 5분마다 |

#### 자동 기록 시작
| 요소 | 설명 |
|---|---|
| 설명 | "앱 실행 시 자동으로 데이터 저장 시작" |
| 토글 | ON/OFF |

#### 캘리브레이션
| 요소 | 설명 |
|---|---|
| 섹션 타이틀 | "캘리브레이션" |
| 설명 | "정확한 소음 측정을 위한 보정 설정" |
| 안내 문구 | "앱 플로그처럼 기준에 맞도록 데이터를 맞춰주세요" |
| 캘리브레이션 시작 버튼 | 기기 연결 시 활성화 (미연결 시 비활성) |
