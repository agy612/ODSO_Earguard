# Earguard 앱 기획서 — 07. 커뮤니티

---

> 커뮤니티 탭은 3개 서브탭(지도 / 목록 / 필터)과 하위 기능(소음 제보, 조용한 출근길 찾기)으로 구성.

---

## 7-1. 커뮤니티 공통 헤더

| 요소 | 설명 |
|---|---|
| 타이틀 | "커뮤니티" |
| 서브 텍스트 | "우리 동네 소음 정보를 공유하고 함께 관리해요" |
| 검색 바 | 장소명 / 내용 검색 |
| 서브탭 | 지도 / 목록 / 필터 |
| "+ 게시글" 버튼 | 소음 제보하기 팝업 오픈 |

---

## 7-2. 지도 탭

> **참조 화면**: screen_27

### 화면 목적
커뮤니티 사용자들이 제보한 소음 측정 위치를 지도 위에 핀으로 시각화.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ 커뮤니티                                 │
│ [🔍 장소명 / 내용 검색              ]    │
│ [지도] [목록] [필터]      [+ 게시글]     │
├─────────────────────────────────────────┤
│                                         │
│         [ 지도 뷰 — 소음 핀 표시 ]       │
│           [핀] [핀] [3] [핀]            │
│                                         │
│                        [내 위치 버튼]   │
├─────────────────────────────────────────┤
│  범례: 안전(~30dB) / 주의(~80dB) / 위험  │
│  [조용한 출근길 찾기 >]                  │
└─────────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 지도 뷰 | 지도 위 소음 핀 표시 |
| 소음 핀 구분 | 안전(30dB 이하) / 주의(30~80dB) / 위험(80dB 이상) |
| 핀 탭 시 | 해당 위치 소음 정보 팝업 (장소명, dB, 제보자, 제보 시간) |
| 범례 | 핀 색상 기준 안내 |
| 내 위치 버튼 | 현재 내 위치로 지도 이동 |
| 조용한 출근길 찾기 버튼 | 해당 기능 화면으로 이동 |

---

## 7-3. 목록 탭

> **참조 화면**: screen_28

### 화면 목적
커뮤니티 소음 제보 게시글 목록 조회.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ 커뮤니티                                 │
│ [지도] [목록] [필터]      [+ 게시글]     │
│                              [정렬 ▾]   │
├─────────────────────────────────────────┤
│  [소음 지역]  닉네임     5분 전           │
│  스타벅스 강남동   78dB                  │
│  "공사 소음이 심해요"                    │
│  ♡ 12   💬 3                           │
├─────────────────────────────────────────┤
│  [조용한 곳]  닉네임     1시간 전         │
│  한강 공원     42dB                      │
│  "산책하기 좋은 조용한 곳"               │
│  ♡ 25   💬 7                           │
│  ...                                    │
└─────────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 서브탭 | 목록 선택 상태 |
| 정렬 옵션 | 최신순 / 소음 높은 순 등 |
| 게시글 카드 리스트 | 스크롤 가능 |

#### 게시글 카드 항목
| 요소 | 설명 |
|---|---|
| 소음 유형 배지 | "소음 지역" / "조용한 곳" 등 구분 |
| 프로필 아이콘 | 익명 또는 닉네임 |
| 장소명 | 제보 위치 |
| 소음 수치 | dB 표시 |
| 제보 내용 | 짧은 텍스트 설명 |
| 제보 시간 | 상대 시간 (예: "5분 전") |
| 좋아요 수 | |
| 댓글 수 | |
| 탭 시 동작 | 게시글 상세 화면으로 이동 |

---

## 7-3-1. 게시글 상세 화면

### 화면 목적
게시글 전체 내용 확인, 댓글 참여, 지도에서 위치 확인.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ [←]                        [소음 지역]  │  ← 뒤로가기 + 게시글 유형 배지
├─────────────────────────────────────────┤
│  [프로필 아이콘]  닉네임     2025.04.10  │
│                                         │
│  스타벅스 강남동                          │
│  78dB · 카페 · 30분 전                   │
│                                         │
│  "오전 출근 시간대 공사 소음이 심합니다.   │
│   이어폰 없이는 통화 불가 수준이에요."     │
│                                         │
│  [첨부 사진 (있는 경우)]                 │
│                                         │
│  [지도에서 위치 보기 >]                  │
├─────────────────────────────────────────┤
│  ♡ 좋아요 12           💬 댓글 3        │
├─────────────────────────────────────────┤
│  댓글 (3)                                │
│  [아이콘] 닉네임   "저도 오늘 지나갔는데" │
│                    30분 전               │
│  [아이콘] 닉네임   "공사 언제까지인지..." │
│                    1시간 전              │
├─────────────────────────────────────────┤
│  [댓글을 입력하세요              ] [전송] │
├─────────────────────────────────────────┤
│  [신고하기]                              │
└─────────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 뒤로가기 | 목록 탭으로 복귀 |
| 게시글 유형 배지 | "소음 지역" / "조용한 곳" |
| 작성자 정보 | 프로필 아이콘 + 닉네임 + 작성일 |
| 장소 정보 | 장소명 + 측정 dB + 카테고리 + 상대 시간 |
| 본문 | 제보 텍스트 전체 |
| 첨부 사진 | 있는 경우 표시, 탭 시 전체화면 |
| 지도에서 위치 보기 | 커뮤니티 > 지도 탭으로 이동하며 해당 핀 포커스 |
| 좋아요 | 탭 시 즉시 반영 (로그인 사용자만) |
| 댓글 목록 | 최신순 정렬 |
| 댓글 입력 | 로그인 사용자만 가능, 미로그인 시 로그인 안내 |
| 신고하기 | 부적절한 게시글 신고 → 확인 팝업 → 처리 안내 토스트 |

## 7-4. 필터

> **참조 화면**: screen_29

### 화면 목적
커뮤니티 목록을 조건에 따라 필터링.

### UI 구성 (바텀시트 형태)
| 요소 | 설명 |
|---|---|
| 카테고리 필터 | 전체 / 카페·식당 / 공원·자연 / 주거지역 / 교통 / 기타 |
| 정렬 | 최신순 / 소음 높은 순 / 좋아요 많은 순 |
| 소음 정도 슬라이더 | dB 범위 설정 (예: 0dB ~ 100dB) |
| 기간 | 오늘 / 이번 주 / 이번 달 / 전체 |
| 초기화 버튼 | 모든 필터 초기화 |
| 적용하기 버튼 | 필터 적용 후 목록으로 이동 |

## 7-5. 소음 제보하기 팝업

> **참조 화면**: screen_30

### 화면 목적
현재 위치의 소음 정보를 커뮤니티에 제보.

---

### 화면 레이아웃

```
┌──────────────────────────────────────┐
│  소음 제보하기                         │
│  위치: [현재 내 위치 자동 입력      ]  │
│  소음: [XX dB 자동 입력             ]  │
│  카테고리: [카페 ▾]                   │
│  내용: [짧은 설명 (선택)           ]  │
│  사진: [첨부 (선택)]                  │
│  [제보하기]                           │
└──────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 팝업 타이틀 | "소음 제보하기" |
| 위치 입력 | 현재 내 위치 자동 입력 (수정 가능) |
| 소음 수치 | 현재 측정된 dB 자동 입력 (수정 가능) |
| 카테고리 선택 | 장소 유형 선택 (카페, 도로, 공원 등) |
| 내용 입력 | 짧은 설명 텍스트 (선택) |
| 사진 첨부 | 선택 (없어도 제보 가능) |
| 제보하기 버튼 | 제출 |

---

## 7-6. 조용한 출근길 찾기

> **참조 화면**: screen_25 (경로 탐색), screen_26 (경로 추적 중)

### 화면 목적
현재 위치 기준 소음이 낮은 추천 경로를 제공하고 실시간 경로 추적.

---

### 7-6-1. 경로 탐색 화면 (screen_25)

---

#### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│  조용한 출근길 찾기                       │
│  70dB 이하 구간으로 이동하세요            │
│                                         │
│           68dB                          │
│        [안전 구간]                       │
│                                         │
│  [측정 시작하기]                         │
├─────────────────────────────────────────┤
│  추천 조용한 경로                         │
│  ┌──────────────────────────────────┐   │
│  │ 공원길 경로   1.2km   62dB  15분  │   │
│  │ 현재 대비 -4dB   [이 경로로 가기] │   │
│  └──────────────────────────────────┘   │
├─────────────────────────────────────────┤
│  소음 안전 팁: "이어폰 볼륨을 낮추세요"  │
└─────────────────────────────────────────┘
```

---

#### UI 구성
| 요소 | 설명 |
|---|---|
| 타이틀 | "조용한 출근길 찾기" |
| 서브 텍스트 | "70dB 이하 구간으로 이동하세요" |
| 현재 소음 수치 | 크게 표시 (예: 68dB) |
| 소음 상태 배지 | "안전 구간" / "위험 구간" |
| 측정 시작하기 버튼 | 경로 추적 시작 |
| 추천 조용한 경로 섹션 | |

#### 추천 경로 카드
| 요소 | 설명 |
|---|---|
| 경로명 | 예: 공원길 경로 |
| 거리 | 예: 1.2km |
| 평균 소음 | 예: 62dB |
| 소요 시간 | 예: 15분 |
| 소음 감소 배지 | 현재 대비 감소 dB (예: -4dB) |
| "이 경로로 가기" 버튼 | 해당 경로로 내비게이션 시작 |

#### 팁 섹션
| 요소 | 설명 |
|---|---|
| 소음 안전 팁 | 현재 소음 상황에 맞는 행동 가이드 짧은 텍스트 |

---

### 7-6-2. 경로 추적 중 화면 (screen_26)

---

#### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│           72dB                          │
│        [위험 구간]                       │
│  경과: 00:12:34   이동: 0.8km            │
│                                         │
│  추천 경로 (실시간 갱신)                 │
│  [공원길 경로  62dB  -4dB]              │
│                                         │
│  [추적 중단]                            │
│─────────────────────────────────────────│
│  ▲ 경로 추적 중입니다                    │  ← 하단 토스트
└─────────────────────────────────────────┘
```

---

#### UI 구성

| 요소 | 설명 |
|---|---|
| 현재 소음 수치 | 실시간 갱신 |
| 소음 상태 배지 | |
| 경과 시간 | 측정 시작 후 경과 시간 |
| 이동 거리 | 이동한 거리 |
| 추적 중단 버튼 | 탭 시 확인 팝업 → 중단 |
| 추천 경로 리스트 | 동일 표시 (실시간 소음 수치 갱신) |
| 하단 토스트 | "▲ 경로 추적 중입니다" |

### 데이터 / 연동
