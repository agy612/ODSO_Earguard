# Earguard 앱 기획서 — 04. 기록

---

> 기록 탭은 4개의 서브 탭으로 구성.
> 공통 헤더: "나의 기록" + 서브탭 (소음 할당량 / 위치 기록 / 행동 분석 / 정밀 검사)

---

## 4-1. 소음 할당량 탭

> **참조 화면**: screen_13

### 화면 목적
일별/주간/월간 소음 에너지 사용 이력을 차트와 리스트로 조회.
홈 > 일일 소음 할당량 카드 탭 시 이 화면으로 진입.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ 나의 기록                                │  ← 공통 헤더
│ [소음할당량] [위치기록] [행동분석] [정밀검사] │  ← 서브탭
├─────────────────────────────────────────┤
│  [일별] [주간] [월간]                    │  ← 기간 탭
├─────────────────────────────────────────┤
│  소음 할당량 / 평균 사용 XX%             │
│                                         │
│  [ 바 차트: 기간별 소음 에너지 사용률 ]   │
│   (일별: 시간대 / 주간: 요일 / 월간: 일) │
│                                         │
│  범례: 안전(70%↓) / 주의(70~90%) / 위험(90%↑) │
├─────────────────────────────────────────┤
│  [← 이전]                     [다음 →]  │
├─────────────────────────────────────────┤
│  2025.01.01   사용 XX%   잔여 XX%       │
│  2025.01.02   사용 XX%   잔여 XX%       │
│  ...                                    │
└─────────────────────────────────────────┘
```

---

### UI 구성
| 요소 | 설명 |
|---|---|
| 서브탭 | "소음 할당량" 선택 상태 |
| 기간 탭 | 일별 / 주간 / 월간 전환 |
| 차트 헤더 | 선택 기간 소음 할당량 평균 사용률 |
| 바 차트 | 기간별 소음 에너지 사용률 (%) |
| 차트 범례 | 안전(70%↓) / 주의(70~90%) / 위험(90%↑) 구분 |
| 기간 전환 버튼 | 이전/다음 기간 조회 |
| 기록 리스트 | 날짜 / 소음 에너지 사용량 % / 잔여량 % |

### 데이터 / 연동
- ⚠️ **일일 소음 할당량 계산식 ODSO 제공 예정** (안전 범위 100 기준 누적 계산식)
  - 리스트의 소음 에너지 사용량 % / 잔여량 % 표현 방식도 해당 계산식 수신 후 확정
- 측정 기기 또는 스마트폰 마이크로 수집된 소음 데이터 기반

## 4-2. 위치 기록 탭

> **참조 화면**: screen_14, screen_15

### 화면 목적
사용자가 방문한 장소별 소음 노출 이력 조회 및 위험 소음 장소 알림.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ 나의 기록                                │
│ [소음할당량] [위치기록] [행동분석] [정밀검사] │
├─────────────────────────────────────────┤
│  기록된 장소 N곳  위험 N곳  안전 N곳     │  ← 요약 카드
├─────────────────────────────────────────┤
│  [🔍 장소 검색                        ] │
│  [전체] [위험] [안전]      [정렬 옵션 ▾] │
├─────────────────────────────────────────┤
│  ⚠️ 2곳의 장소에서 위험한 소음            │  ← 위험 배너 (해당 시)
│  (85dB 이상)이 측정되었어요.             │
│  가능한 이곳을 피하거나 짧은 시간만        │
│  머무르세요.                             │
│  [스타벅스 강남동 ×] [2호선 강남역 ×]    │  ← 위험 장소 태그
├─────────────────────────────────────────┤
│  [●] 스타벅스 강남동     73~78dB  카페   │
│       체류 2h  · 어제         [위험 배지] │
│  [●] 2호선 강남역        85~92dB  지하철 │
│       체류 45m · 3일 전       [위험 배지] │
│  ...                                    │
└─────────────────────────────────────────┘
```

**장소 상세 바텀시트 (screen_15)**
```
┌─────────────────────────────────────────┐
│  장소명                                  │
│  평균 XXdB   [위험/주의/안전]            │
│  방문 시간: XX:XX ~ XX:XX               │
│  [커뮤니티에 소음 제보]                   │
│  [이 장소 알림 해제]    [기록 삭제]       │
└─────────────────────────────────────────┘
```

---

### UI 구성

#### 요약 카드
| 요소 | 설명 |
|---|---|
| 기록된 장소 수 | 위치 기록이 남은 총 장소 수 |
| 위험 장소 수 | 위험 소음(85dB 이상)이 측정된 장소 수 |
| 안전 장소 수 | 안전 기준 이하 장소 수 |

#### 검색 및 필터
| 요소 | 설명 |
|---|---|
| 장소 검색 바 | 장소명 검색 |
| 필터 탭 | 전체 / 위험 / 안전 |
| 정렬 옵션 | 최신순 / 소음 높은 순(dB) |

#### 위험 소음 경고 배너
| 요소 | 설명 |
|---|---|
| 경고 아이콘 | 위험 소음 장소 알림 강조 |
| 경고 메시지 | "N곳의 장소에서 위험한 소음(85dB 이상)이 측정되었어요. 가능한 이곳을 피하거나 짧은 시간만 머무르세요." |
| 위험 장소 태그 | 해당 장소명 태그 목록 — 탭 시 해당 장소 바텀시트로 바로 이동 |
| 노출 조건 | 위험 배지 장소가 1곳 이상일 때만 표시 |

#### 장소 리스트
| 요소 | 설명 |
|---|---|
| 장소 아이콘 | 소음 수준 표시 |
| 장소명 | 예: 스타벅스 강남동 |
| 소음 수치 범위 | 예: 73~78dB |
| 장소 카테고리 | 카페, 지하철, 쇼핑몰 등 |
| 총 체류 시간 | 예: 2h |
| 방문 시점 | 상대 시간 (예: 어제, 3일 전) |
| 위험/안전 배지 | |

#### 장소 상세 바텀시트 (screen_15)
| 요소 | 설명 |
|---|---|
| 장소명 | |
| 소음 수치 | 평균 dB |
| 소음 상태 | 위험/주의/안전 |
| 방문 시간 | |
| 위치 공유 버튼 | 커뮤니티에 소음 제보 연동 |
| 알림 해제 버튼 | 해당 장소 위험 알림 끄기 |
| 삭제 버튼 | 해당 기록 삭제 |

## 4-3. 행동 분석 탭 (AI)

> **참조 화면**: screen_16

> 💡 추가 기획 여지 있음 — AI 분석 기준(어떤 데이터를 어떻게 분석할 것인지), 인사이트 카드 구성 방식, 행동 가이드 개인화 로직 등 ODSO와 협의 및 구체화 필요

### 화면 목적
사용자의 주간 소음 관리 패턴을 AI가 분석하여 인사이트 및 행동 개선 가이드 제공.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ 나의 기록                                │
│ [소음할당량] [위치기록] [행동분석] [정밀검사] │
├─────────────────────────────────────────┤
│  주간 관리 성공!                          │  ← AI 인사이트 카드
│  이번 주 평균 소음 기준 달성               │
├─────────────────────────────────────────┤
│  완벽한 한 주!                           │  ← 상세 인사이트 카드
│  모든 날 건강한 소음 환경 유지             │
├─────────────────────────────────────────┤
│  이렇게 해보세요!                         │  ← 행동 가이드
│                                         │
│  [아이콘] 출퇴근 시 조용한 경로 찾기       │
│           → +10 dB 감소                  │
│  [아이콘] 이어폰 볼륨 60% 이하 유지        │
│           → 청력 보호 효과               │
│  ...                                    │
└─────────────────────────────────────────┘
```

---

### UI 구성

#### AI 인사이트 카드
| 요소 | 설명 |
|---|---|
| 긍정 인사이트 카드 | "주간 관리 성공!" — 이번 주 평균 소음 기준 달성 메시지 |
| 긍정 상세 카드 | "완벽한 한 주!" — 모든 날 건강한 소음 환경 유지 등 |

#### 이렇게 해보세요 (행동 가이드)
| 요소 | 설명 |
|---|---|
| 섹션 타이틀 | "이렇게 해보세요!" |
| 행동 가이드 항목 리스트 | 각 항목: 아이콘 + 제목 + 효과 설명 |

**가이드 항목 예시 (AI 생성)**
- 출퇴근 시 조용한 경로 찾기 → +10 dB 감소
- 이어폰 볼륨 60% 이하 유지 → 청력 보호 효과
- 주말엔 공원이나 자연에서 휴식 → 소음 노출 30% 감소
- 충분한 수면으로 청력 회복 → 청력 민감도 회복

## 4-4. 정밀 자기검사 탭

> **참조 화면**: screen_17, screen_18

### 화면 목적
청력 검사 이력 조회 및 4가지 정밀 자기검사 항목 제공.

---

### 화면 레이아웃

```
┌─────────────────────────────────────────┐
│ 나의 기록                                │
│ [소음할당량] [위치기록] [행동분석] [정밀검사] │
├─────────────────────────────────────────┤
│  청력 나이: XX세                          │
│  정기적인 검사로 청력 건강을 관리하세요     │
│  마지막 검사: YYYY.MM.DD                 │
├─────────────────────────────────────────┤
│  [순음 청력 검사           →]            │
│   다양한 주파수에서의 청력 기준 측정       │
│  [소음의 청취 능력         →]            │
│   소음 환경에서 음성을 고르는 능력         │
│  [소음 노출 설문           →]            │
│   일상 소음에 노출되는 정도 파악          │
│  [청력 건강 설문           →]            │
│   전반적인 청력 상태 파악                │
└─────────────────────────────────────────┘
```

**순음 청력 검사 시작 팝업 (screen_18)**
```
┌──────────────────────────────┐
│  [헤드셋 아이콘]               │
│  순음 청력 검사                │
│  헤드셋을 끼거나 조용한 곳에서  │
│  버튼을 눌러 시작하세요         │
│  [시작하기]                   │
└──────────────────────────────┘
```

---

### UI 구성

#### 상단 요약
| 요소 | 설명 |
|---|---|
| 청력 나이 | 현재 청력 나이 수치 (예: 28세) |
| 안내 텍스트 | "정기적인 검사로 청력 건강을 관리하세요" |
| 마지막 검사일 표시 | |

#### 검사 항목 리스트
| 항목 | 설명 | 이동 |
|---|---|---|
| 순음 청력 검사 | 다양한 주파수에서의 청력 기준 측정 | 검사 시작 팝업 → 검사 화면 |
| 소음의 청취 능력 | 소음 환경에서 음성을 고르는 능력 | 검사 시작 팝업 → 검사 화면 |
| 소음 노출 설문 | 일상 소음에 노출되는 정도 파악 | 설문 화면 |
| 청력 건강 설문 | 전반적인 청력 상태 파악 | 설문 화면 |

#### 순음 청력 검사 시작 팝업 (screen_18)
| 요소 | 설명 |
|---|---|
| 헤드셋 아이콘 | |
| 타이틀 | "순음 청력 검사" |
| 안내 문구 | "헤드셋을 끼거나 조용한 곳에서 버튼을 눌러 시작하세요" |
| 시작하기 버튼 | 검사 화면으로 이동 |

### 데이터 / 연동
- ⚠️ **순음 청력 검사 / 소음의 청취 능력: 온보딩 Step 2와 동일한 모두의보청기 소스코드 사용** (소스 ODSO 전달 예정)
- ⚠️ **청력 나이 계산식: ODSO 전달 예정**
- 소음 노출 설문 / 청력 건강 설문 항목: ODSO 제공 예정
- 검사 결과는 날짜별 이력으로 저장, 청력 나이 재계산에 반영
